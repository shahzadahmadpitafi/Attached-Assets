# Replit Prompt: Qanzak Global Properties - Admin Dashboard

Create a comprehensive admin dashboard for Qanzak Global Properties that allows the business owner to manage all website content, properties, inquiries, and settings. This should integrate with the existing frontend website.

## Business Information
- **Business Name:** Qanzak Global Properties
- **Phone:** +92 331 1479800
- **Email:** info@qanzakglobal.com
- **Location:** House 66, F-11/1, Islamabad, Pakistan

## Technical Stack Requirements
- **Backend:** Node.js with Express.js OR Next.js API routes
- **Database:** MongoDB (MongoDB Atlas) OR Firebase OR Supabase
- **Authentication:** JWT tokens with secure login system
- **Frontend Admin Panel:** React with modern UI library (Tailwind CSS + shadcn/ui OR Material-UI)
- **File Upload:** Cloudinary OR Firebase Storage OR AWS S3 for property images
- **API Integration:** RESTful API connecting frontend website to backend

### 12. API ENDPOINTS NEEDED

#### Authentication
- POST /api/auth/login
- POST /api/auth/logout
- POST /api/auth/forgot-password
- POST /api/auth/reset-password
- GET /api/auth/verify-token

#### Properties
- GET /api/properties (list all with filters)
- GET /api/properties/:id (get single property)
- POST /api/properties (create new)
- PUT /api/properties/:id (update)
- DELETE /api/properties/:id (delete)
- PATCH /api/properties/:id/status (update status)
- GET /api/properties/featured (get featured properties)

#### Inquiries
- GET /api/inquiries (list all)
- GET /api/inquiries/:id (get single)
- POST /api/inquiries (create - from website form)
- PUT /api/inquiries/:id (update)
- DELETE /api/inquiries/:id (delete)
- PATCH /api/inquiries/:id/status (update status)

#### Media
- POST /api/media/upload (upload images)
- GET /api/media (list all)
- DELETE /api/media/:id (delete)

#### Team
- GET /api/team (list all)
- POST /api/team (create)
- PUT /api/team/:id (update)
- DELETE /api/team/:id (delete)

#### Settings
- GET /api/settings (get all settings)
- PUT /api/settings (update settings)

#### Analytics
- GET /api/analytics/properties (property stats)
- GET /api/analytics/inquiries (inquiry stats)
- GET /api/analytics/revenue (revenue stats)
